fairseq2.setup_fairseq2()[source]¶

Sets up fairseq2 by initializing its global DependencyContainer.

As part of the initialization, this function also registers external objects with the container via setuptools’ entry-point mechanism. See Runtime Extensions for more information.

Important

This function must be called before using any of the fairseq2 APIs.
