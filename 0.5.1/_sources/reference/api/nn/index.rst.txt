fairseq2.nn
===========

.. currentmodule:: fairseq2.nn

The neural network module provides building blocks and utilities for creating
transformer-based models and other neural network architectures. It includes
the unified batch layout system, core neural network layers, and various
utilities for efficient neural network operations.
